## 3. 변수
   `int a` : 숫자 정수를 저장할 수 있는 a라는 변수를 만든다.

   `a=10` : 변수 a에 값 10을 저장한다.
- 변수 초기화 : 선언한 변수에 처음으로 값을 대입해서 저장하는 것

   **변수는 반드시 초기화해야한다.**
   **초기화하지 않으면 이상한 값이 출력될 수 있음**
   ### 변수 타입
  - int : 정수
  - float : 실수
  - float은 숫자 뒤에 f 써준다.
  - double:실수

     *float보다 double이 더 큰 범위 표현 가능*
  - boolean : 불리언 
  - char : 문자 하나 
  - String : 문자열

- 클래스는 첫 글자 대문자, 나머지는 모두 첫 글자 소문자 => 낙타표기법
- 예외 
  - 상수 : 모두 대문자, 언더바로 구분 
  - 패키지 : 모두 소문자

## 4.연산자
- 연산자 : 연산기호 
- 피연산자 : 연산대상 
- 산술연산자 :  +,-,*,/(나누기),%(나머지)
  - **int형끼리의 계산은 소수점 이하 표현이 안된다.**
  - **0으로 나눌 수는 없다.**
- 증감연산자 ++, --
- 비교연산자 ==, !=, >,<,>=,<=
- 논리연산자 &&(and), ||(or), !(not)
- 대입연산자 =, +=, -=, *=, /=, %=
- 삼항연산자 ?:

  - **괄호가 우선순위가 제일 높고 =(대입연산자)가 가장 낮다.**
  - **괄호를 사용하여 직관적으로 보이도록 사용**


- a = a+1
- b=++a : a를 먼저 증가시키고 대입한다.
- b=a++ : a를 먼저 대입하고 증가시킨다.
- a = a-1
- b=--a : a를 먼저 감소시키고 대입한다.
- b=a-- : a를 먼저 대입하고 감소시킨다.

- **증감연산자 단독으로 사용하면 둘의 결과는 같다.**


  **>:크다**

  **>= : 크거나 같다**
- == : 동등성
- != : 불일치

- < : 작다

- <= : 작거나 같다
- =과 ==은 다르다
- =은 대입연산자, ==은 동등한지 확인
- 문자열 비교는 ==이 아니라 .equals()

## 5.조건문
- if : 특정 조건이 참일때만 실행하는 조건문
- else : if문에서 만족하는 조건이 없을 때 실행하는 코드를 제공
- else if : 불필요한 조건 검사를 피할 수 있다
**if문에 여러 조건이 있다고 항상 if-else로 묶어서 사용할 수 있는 건 아니다.**
**조건이 서로 영향을 주지 않고 각각 수행해야 하는 경우 else if문을 사용하면 안되고 대신에 여러 if문을 분리해서 사용**

## 6.반복문
- 특정 코드를 반복해서 실행
**while, do-while, for**
- do-while : 조건에 상관없이 무조건 한 번은 코드를 실행한다.
- break : 반복문 즉시 종료
- continue : 반복문의 나머지 부분을 건너뛰고 다음 반복으로 진행
