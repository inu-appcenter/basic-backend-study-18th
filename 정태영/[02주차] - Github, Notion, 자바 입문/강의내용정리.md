## 변수

### 변수의 선언과 초기화

```java
// case 1:
int a; 
a = 10;
// case 2:
int a = 10;
```

- `int` 형태의 값을 저장할 수 있는 `a`라는 데이터 저장소 생성
- `=` 은 오른쪽에 있는 값을 왼쪽에 저장한다는 의미
- 변수를 **초기화하지 않고** 사용하면 **컴파일 에러 발생**🚫 ← 다른 곳에서 쓰던 값(쓰레기 값)이 존재하기 때문

### 변수 값 읽기

```java
System.out.println(a); // 10을 출력
```

- 변수 `a` 의 값을 읽고 출력

### 꿀팁

```java
// System.out.println(a); // 주석을 풀면 컴파일 오류 발생
```

- 에러 확인 후 해당 라인에 주석 적용

### 변수 타입

| 타입 | 값 | 메모리 공간 | 범위 |
| --- | --- | --- | --- |
| 정수형 | `byte`  | 1 byte | -128 ~ 127 |
|  | `short` | 2 byte | -32,768 ~ 32,767 |
|  | `int`✅ | 4 byte | -2,147,483,648 ~ -2,147,483,647 |
|  | `long` | 8 byte | -9,223,372,036,854,775,808 ~ -9,223,372,036,854,775,807 |
| 실수형 | `float` | 4 byte | 7자리 정밀도 |
|  | `double`✅ | 8 byte | 15자리 정밀도 |
| 문자 | `char` | 1 byte | 문자 하나 |
| 문자열 | `String`✅ | 길이에 따라 | 문자 여러개 |
| 불린형 | `boolean`✅ | 1 byte | `true`, `false` |
- 선언한 타입과 값의 타입이 일치하지 않으면 오류 발생🚫
- 정수형은 웬만해서 `int` 만 사용, 길면 `long`
- 실수형도 `double` 만 사용
- `char` 보다 그냥 `String` 사용

### 리터럴

```java
int a = 100; // 정수 리터럴
String e = "Hello AppCenter"; // 문자열 리터럴
```

- 개발자가 작성된 고정된 값을 의미

### 리터럴 타입 지정

- 정수 리터럴은 기본적으로 `int` 로 인식 → 큰 정수 리터럴은 `L` 을 붙여 `long` 임을 표시해야 함
- 실수는 `double` 로 인식 → `float` 로 표현하고 싶으면 `F` 를 붙임(**실무에서 쓰지 않음**)

### 변수 명명 규칙

1. 변수 이름 숫자 ❌ → 숫자 포함은 가능
2. 공백 ❌
3. 예약어 ❌ → ex) `int` , `class` , `static` 
4. 영문자✅, 숫자✅, 달러 기호✅, 밑줄✅

### 국룰

이름은 의미있고, 그 용도를 명확하게 설명해야 함

```java
int a, b; // 용도를 알 수 없음
int studentCount, maxScore; // 변수 이름으로 용도 파악 가능
```

1. **변수** : 낙타 표기법 (Camel Case)
    - 변수 이름을 지을 때 단어 여러개로 구성되어 있다면 대문자로 구분하여 표기
    - ex) `orderdetail` → `orderDetail`
2. **클래스** : 첫 글자 대문자 + 낙타 표기법
    - ex) `Person` , `OrderDetail`
3. **상수 :** 모두 대문자 + 언더바 구분
    - ex) `USER_LIMIT`
4. **패키지 :** 모두 소문자
    - ex) `org.spring.boot`

## 연산자

### 산술 연산자와 문자열 더하기

```java
package operator;

public class Operator1 {
    public static void main(String[] args) {
        int num1 = 10;
        int num2 = 3;

        int sum = num1 + num2;
        // num1, num2 : 피연산자
        // + : 산술 연산자
        System.out.println("num1 + num2 = " + sum); // 출력 : num1 + num2 = 13

        int div = num1 / 0; // 0으로 나누면 오류 발생 
        
        System.out.println("Hello " + "world"); // 문자열 더하기
        System.out.println("num1 / 5  = " + (num1/5)); // 문자열과 숫자 더하기
    }
}
```

- 인텔리제이에서 `soutv`를 타이핑하면 `System.out.println(”num1 = ” + num1 );`  이런 식으로 `“num1 = ”` 를 자동 삽입해줌

### 연산자 우선순위

```java
sum1 = 1 + 2 * 3; // 출력 : 7
sum2 = (1 + 2) * 3; // 출력 : 9
```

- 연산자의 우선순위에 따라 출력이 달라짐
- 헷갈릴 경우 `()` 괄호를 사용

### 증감 연산자

```java
package operator;

public class Operator2 {
    public static void main(String[] args) {
        int a = 1;
        int b = 0;

        /* 전위 증감 연산자 */
        b = ++a; // a값을 먼저 증가시키고, 그 값을 b에 대입
        System.out.println("a = " + a + ", b = " + b);

        a = 1;
        b = 0;

        /* 후위 증감 연산자 */
        b = a++; // a값을 먼저 b에 대입하고, a값을 증가시킴
        System.out.println("a = " + a + ", b = " + b);
    }
}

```

## 조건문

### if, if-else, else

```java
if(condition1){
	// 조건 1이 참일 때 실행되는 코드
}
else if(condition2){
	// 조건 1이 거짓이고, 조건 2가 참
}
else if(condition3){
	// 조건 1,2가 거짓이고, 조건 3이 참
}
else {
	// 조건 1,2,3 모두 거짓
}
```

### switch

```java
switch(조건식){
	case value1:
		// 실행 코드
		break;
	case value1:
		// 실행 코드
		break;
	default:
		// 실행 코드
}
```

- 반드시 `break` 사용해야 함

### 개선된 switch

자바 14 부터 지원

```java
int grade = 2;

int coupon = switch(grade){
		case 1 -> 1000;
		case 2 -> 2000;
		case 3 -> 3000;
		default -> 500;
};
// 결과 : coupon => 2000
```

## 반복문

### while

```java
int count = 0;
while(count < 3){
	count++;
	System.out.println("현재 숫자는 : " + count);
}
/* 출력 결과
현재 숫자는 : 1
현재 숫자는 : 2
현재 숫자는 : 3
*/
```

### do while

```java
int count = 0;
do{
	count++;
	System.out.println("현재 숫자는 : " + count);
}while(count < 3)
/* 출력 결과
현재 숫자는 : 1
현재 숫자는 : 2
현재 숫자는 : 3
*/
```

### for

주로 반복 횟수가 정해져 있는 경우 사용

```java
for(int count = 0; count < 3; count++){
	System.out.println("현재 숫자는 : " + count);
}
/* 출력 결과
현재 숫자는 : 1
현재 숫자는 : 2
현재 숫자는 : 3
*/
```
