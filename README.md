## 커리큘럼

앱센터 베이직의 커리큘럼은 다음과 같습니다.

1. 매주 주어진 강의와 미션을 정해진 날짜 전까지 수행합니다.
2. 미션 수행 결과와 소스코드를 Github과 Notion에 제출합니다.
3. 지난 주에 제출한 자료와 추가적인 지식을 기술 Blog에 작성 및 공유합니다.
